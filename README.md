# Vegetable Image Classification (Streamlit)

Streamlit app that classifies vegetable images into 15 classes using a PyTorch model.
The model is downloaded automatically from Google Drive when the app runs.

## How to run locally
1. Install requirements: `pip install -r requirements.txt`
2. Run: `streamlit run app.py`

## Deploy
Push repo to GitHub and deploy using Streamlit Cloud (instructions in main README).
